Módulo 1: Formularios inteligentes

Introducción al uso de KoboToolbox para captura de información geoespacial en campo orientada a monitoreo ambiental, restauración de ecosistemas y recopilación de datos territoriales.

Herramienta principal: KoboToolbox

Objetivo del módulo

Aprender los principios básicos para diseñar formularios digitales y recopilar información geográfica desde dispositivos móviles o navegadores web.

Contenidos

  • Introducción a KoboToolbox
  • Creación de formularios digitales
  • Captura de coordenadas geográficas
  • Recolección de fotografías en campo
  • Exportación de datos CSV y GeoJSON
  • Integración con SIG

Ejercicio práctico

Crear un formulario básico para captura de puntos de monitoreo ambiental incluyendo:

  • Nombre del sitio
  • Coordenadas GPS
  • Tipo de ecosistema
  • Fotografía
  • Observaciones

Archivo XLSForm para KoboToolbox

Descargue el archivo XLSForm en formato Excel para importarlo directamente en KoboToolbox y crear el formulario de monitoreo ambiental.

Resultado esperado

El participante podrá importar un XLSForm en KoboToolbox, crear un formulario básico de monitoreo ambiental y generar información geoespacial lista para ser utilizada en QGIS, ArcGIS Pro o Google Earth Engine.
