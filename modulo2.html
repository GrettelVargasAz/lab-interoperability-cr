Contenidos

  • Visualización y manejo de capas geoespaciales
  • Simbología y estilos cartográficos
  • Edición de datos vectoriales
  • Integración de datos provenientes de KoboToolbox
  • Manejo de tablas y atributos
  • Sistemas de coordenadas y proyecciones
  • Creación de mapas temáticos
  • Introducción a interoperabilidad geoespacial
  • Exportación de mapas y datos

Ejercicio práctico

En este ejercicio se trabajará un flujo de interoperabilidad entre KoboToolbox y QGIS para visualizar y actualizar automáticamente los datos recolectados en el Módulo 1.

El objetivo es conectar los formularios digitales creados en KoboToolbox con QGIS utilizando la API de KoboToolbox y Python, permitiendo visualizar la información geoespacial dentro del entorno SIG.

Paso a paso del ejercicio

  • Crear un formulario de monitoreo en KoboToolbox desde el Módulo 1.
  • Capturar información geoespacial en campo utilizando coordenadas GPS y fotografías.
  • Publicar y almacenar los datos recolectados en KoboToolbox.
  • Obtener el TOKEN y UID del formulario mediante la API de KoboToolbox.
  • Configurar el script de interoperabilidad en QGIS utilizando Python y PyQGIS.
  • Conectar QGIS con KoboToolbox mediante solicitudes API en formato GeoJSON.
  • Descargar automáticamente los datos recolectados desde KoboToolbox.
  • Visualizar y actualizar las capas geoespaciales dentro de QGIS.
  • Aplicar simbología y análisis geoespacial sobre los datos recolectados.
  • Exportar la información para su uso posterior en dashboards, WebGIS o Google Earth Engine.

Ejemplo de interoperabilidad KoboToolbox + QGIS

El siguiente ejemplo muestra un flujo básico de interoperabilidad utilizando la API de KoboToolbox y PyQGIS para actualizar automáticamente una capa geoespacial en QGIS. 

from qgis.PyQt.QtWidgets import QAction
from qgis.PyQt.QtGui import QIcon
from qgis.core import QgsVectorLayer, QgsProject

import requests
import os

# CONFIGURACIÓN KOBO

TOKEN = "TOKEN_KOBO"
UID = "UID_FORMULARIO"

URL = f"https://kf.kobotoolbox.org/api/v2/assets/{UID}/data/?format=geojson"

# DESCARGA DE DATOS

response = requests.get(
    URL,
    headers={"Authorization": f"Token {TOKEN}"}
)

# GUARDAR GEOJSON

if response.status_code == 200:

    ruta = os.path.join(
        os.path.expanduser("~"),
        "datos_kobo.geojson"
    )

    with open(ruta, "w", encoding="utf-8") as f:
        f.write(response.text)

    # CARGAR CAPA EN QGIS

    capa = QgsVectorLayer(
        ruta,
        "KoboToolbox",
        "ogr"
    )

    if capa.isValid():

        QgsProject.instance().addMapLayer(capa)

        iface.messageBar().pushSuccess(
            "KoboToolbox",
            "Capa actualizada correctamente."
        )

Flujo de interoperabilidad

El flujo conecta la captura de datos en campo mediante KoboToolbox con QGIS, facilitando la actualización automática de información geoespacial y fortaleciendo procesos de interoperabilidad territorial.

Resultado esperado

El participante podrá utilizar QGIS para visualizar, actualizar y analizar datos recolectados en KoboToolbox mediante un flujo de interoperabilidad basado en API y GeoJSON.
